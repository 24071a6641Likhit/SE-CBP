Frontend (Vite + React) — Quickstart

Prerequisites
- Node.js 18+

Install & run

```bash
cd frontend
npm install
npm run dev
```

The dev server runs on `http://localhost:5173` by default. The backend must be running at `/api` on the same host (or set up a proxy in `vite.config.js`).

Use credentials: `maintainer`, `coordinator`, `teacher`, `student_test` (password `changeme`).
