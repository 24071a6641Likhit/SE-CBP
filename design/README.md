# Design Artifacts

This folder stores architecture and API design assets.

## Files

- `openapi.yaml` - API contract for backend endpoints.
- `events-schema.json` - Event payload/schema reference.
- `sequence-diagrams.mmd` - Mermaid sequence diagrams.
- `DIAGRAMS_STRUCTURAL_BEHAVIORAL.md` - Structural and behavioral diagram notes.
- `College_Event_Diagrams.mdj` - Main StarUML model file.
- `College_Event_Diagrams.broken_backup.mdj` - Backup copy.

## Suggested Workflow

1. Update flow and interactions in sequence diagrams.
2. Reflect endpoint changes in `openapi.yaml`.
3. Keep SRS and implementation consistent with design updates.
