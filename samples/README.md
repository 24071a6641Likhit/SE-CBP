# Sample CSV Files

This folder provides sample data files for import testing.

## Files

- `roster.csv` - Sample student roster.
- `timetable.csv` - Sample timetable with period columns.
- `teachers.csv` - Sample subject-to-teacher mapping.

## Usage

- Use these files to verify import flow and schema validation during local development.
- Modify copies of these files for negative/edge-case validation tests.

## Validation Reference

- Schema definitions are in `schemas/`.
- Import rules are documented in `import/README.md`.
